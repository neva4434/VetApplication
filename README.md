# VetApplication
# Veterinary Management System

## Overview

VetApp is a simple Java console application designed to manage a veterinary clinic.
It allows veterinarians to manage animals, appointments, medical records, and prescriptions.

The application stores data in text files and runs in the console.

---

## Features

* Manage animals (Dog, Cat, Bird)
* Manage pet owners and veterinarians
* Create and track appointments
* Store medical operations
* Create prescriptions with medicines and dosages
* Save and load data from text files

---

## Project Structure

```
src
 ├ animal
 ├ appointment
 ├ chip
 ├ medical
 ├ menu
 ├ people
 ├ prescription
 ├ repository
 └ main
```

Main entry point:

```
src/main/Main.java
```

---

## Data Storage

The application stores information in the following files:

* animals.txt
* appointments.txt
* prescriptions.txt
* medical_history.txt

---

## How to Run

1. Open the project in IntelliJ IDEA
2. Make sure Java SDK is configured
3. Run the file:

```
src/main/Main.java
```
